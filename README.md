# Zakyus

### The tasks Collector

This is a task management web app for creators to manage their tasks and schedule.

It currently being built with the foolwong technologies:

-   Laravel
-   React for the front end
-   Tailwind CSS
-   axios
-   Font awesome
-   and is currently hosted on heroku.
